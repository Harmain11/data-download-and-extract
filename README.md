# Data Download and Extract

## Overview  
This notebook demonstrates how to download a zipped dataset from a URL and extract its contents using Python in a Jupyter Notebook or Google Colab environment.

## Features  
- Download a zip file from a web URL silently  
- Extract the contents of the zip file  
- Prepare data for subsequent analysis  

## Tech Stack  
- Python  
- Jupyter Notebook  
- Google Colab  
- wget (for downloading files)  
- unzip (for extracting files)  

## How to Use  
1. Open the notebook in Jupyter or Google Colab.  
2. Run the cells to download and extract the data.  
3. Proceed with data analysis using the extracted files.
